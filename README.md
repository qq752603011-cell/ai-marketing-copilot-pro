# 🚀 AI Marketing Copilot

企业级 AI 营销内容生成平台

基于 Python + Streamlit + DeepSeek API 开发，帮助电商运营快速生成营销内容，提高内容创作效率。

---

## ✨ 项目功能

### 商品标题生成
自动生成高转化率商品标题

### 核心卖点提炼
自动提炼产品核心竞争优势

### 小红书营销文案
生成种草风格内容

### 短视频口播稿
生成30秒短视频脚本

### 亚马逊 Bullet Point
生成符合亚马逊规范的卖点文案

### AI模型切换
支持：

- DeepSeek Chat
- DeepSeek Reasoner

### 文件导出

支持导出：

- Word (.docx)
- PDF (.pdf)
- Markdown (.md)

### Token统计

实时显示：

- 输入Token
- 输出Token
- 总Token

## 🚀 本地运行

安装依赖：

bash pip install -r requirements.txt 

启动项目：

bash streamlit run app.py 

---

## 🔑 环境变量

创建：

text .streamlit/secrets.toml 

内容：

toml DEEPSEEK_API_KEY="your_api_key"
